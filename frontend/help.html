Gameplay

To start playing enter your name into the input box and press the "Play!" button. Traverse the map, shoot (see below for controlls) skeletons and other players to get points.

Controls

If you have a physical keyboard use WASD or arrow keys to move and press <space> to shoot. If you have a touchscreen, use left half of the screen for movement and right half of the screen to attack in that direction.

Requirements

Archers! Game is written using relatively new browser technologies, hence you need to have the latest, compatible, browser. Here is a list of what is known to work ok:

  • Latest Firefox (desktop & mobile)
  • Latest Chrome (desktop & mobile)
  • Latest Opera (desktop only :( )

On the other hand we've found the following browser to lack some features required for Archers! to work:

  • Internet Explorer, even version 10 is lacking full support for typed arrays
